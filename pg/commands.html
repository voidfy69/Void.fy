Introduction

Amarok is the perfect All In One bot to spice up your server. The bot is community driven and can also be grown with your help.

About this Bot

The bot provides both Antinuke and Automod features with Antilink features at it's core. The bot is hosted 24/7 and you can be reassured about it's uptime.

Technology

The bot is bulit with Javascript using the discord.js library, same with the API the bot uses.

The bot uses it's own API to filter and provide high quality images to your server. With the help of our super fast and flexible API we are able to provide you the best and fastest features.

Getting Started

To get started with our bot is fairly easy and we have done our best to make it simpler for you.

In-bot help

There is also a in-bot gettingstarted command that you can use if you know the installation process.
You also have the option to get help on all the bot commands right through discord with the use of the bot's internal help command.

Important: You must agree to our imagepolicy and matter regarding it will be fairly strict.
Use dh imagepolicy for more information.

Adding the bot

To add the bot to your server click here.

This does needs you to either be the owner of the server or have "manage server" permissions.

Commands

Syntax:
  • < > : Required Argument
  • [ ] : Optional Argument
  • ( ) : Command Aliases

Antinuke

  • Antinuke
    • Description : Let's amarok guard your server with the antinuke module.
    Usage : antinuke enable < your anime >
    Example : antinuke guide
    Note : The bot required permission must be given.
  • Antilink
    • Description : Let's Amarok guard your server from links.
    Usage : antilink < your manga >
    Example : antilink enable
    Note : The bot required permission must be given.
  • Moderation
    • Description : Let's Amarok Give A Mod.
    Usage : clear < your code >
    Example : clear bots
    Aliases: search, id
    Note : The bot required permission must be given.
  • Welcome
    • Description : Let's Amarok Welcome New Members In Your Server.
    Usage : welcome < character name >
    Example : welcome message panel
    Note : The bot required permission must be given.
  • Server
    • Description : extra.
    Usage : extra admin add < link >
    Aliases : extra
